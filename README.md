## 能做什么

| 你想做的事 | EasyTier Native Mac 帮你怎么做 |
| --- | --- |
| 让多台设备互相访问 | 把 Mac、家用电脑、办公室设备和服务器放进同一个 EasyTier 网络。 |
| 快速确认网络是否正常 | 打开应用就能看到当前网络、在线设备和连接状态。 |
| 管理多个使用场景 | 为家庭、工作、远程服务器或临时项目分别保存网络。 |
| 发现网络是否有流量 | 用图表查看上传和下载趋势，不再只凭感觉判断。 |
| 排查连接问题 | 运行记录集中展示，出错时能更快找到线索。 |
| 长时间保持连接 | 应用可以留在菜单栏，平时不打扰，需要时点开查看。 |

## 为什么会有这个项目

EasyTier 很适合把不同设备连接到同一个网络里，但对很多 Mac 用户来说，日常使用更需要的是一个清楚的界面：现在连上了吗？哪些设备在线？速度有没有变化？出错时应该看哪里？

EasyTier Native Mac 想解决的就是这件事：把常见动作放到桌面应用里，让 EasyTier 更适合作为每天都能用的 Mac 网络工具。

## 开始使用

当前项目面向 macOS 14 及以上系统。

推荐从仓库的 [Releases](https://github.com/socoldkiller/easytier-swift/releases) 页面获取安装包。

| 步骤 | 操作 |
| --- | --- |
| 1 | 下载并打开 EasyTier Native Mac。 |
| 2 | 根据 macOS 提示完成必要授权。 |
| 3 | 添加你的网络，检查状态，然后开始连接。 |

> [!IMPORTANT]
> 网络工具可能需要系统权限才能完整工作。首次运行时，请按照 macOS 的提示完成授权。

## 适合谁

- 已经在使用 EasyTier，希望在 Mac 上有一个更直观入口的人。
- 想把家里、公司、服务器或远程设备连在一起的人。
- 希望网络工具能长期运行，但平时不要打扰的人。
- 想用图形界面确认连接状态和设备在线情况的人。
